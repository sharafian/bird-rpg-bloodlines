# Local Development
- [Phaser](https://phaser.io) is the game framework
- [Typescript](https://www.typescriptlang.org/) as the language
- [Webpack](https://webpack.js.org/) ties it all together
- [Yarn](https://yarnpkg.com/) instead of npm

```
yarn
yarn build
yarn start
```
